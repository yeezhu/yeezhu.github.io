Introduction

In this work, we propose a network component called Soft Proposal, which can be inserted after any CNN layer.
The Soft Proposal map iteratively evolves and jointly optimizes with convolutional filters, thus provides objectness information during the end-to-end learning procedure of CNN.
CNN equipped with Soft Proposal can be led to learn better object-centric filters, discover more discriminative visual evidence, and suppress background interference.
